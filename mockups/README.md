# Mockups — throwaway layout exploration

Self-contained HTML (no build step). Open directly in a browser; the RSVP hero is interactive.

- **`studio.html`** — Mockup A: "studio / three-pillar." A scrolling page — manifesto + RSVP hero, an origin/bio strip, then three stacked divisions (mods / apps / tech) with flagship feature cards.
- **`hub.html`** — Mockup C: "game-menu hub." A single-screen "settings menu" (nod to the 2007 origin story) — a left category menu and a right screen that swaps panels in place; RSVP is the default "Story" screen.

Both use the same clean-modern-studio palette and the same real copy, so the comparison is about **information architecture**, not styling. Discard whichever direction loses.
